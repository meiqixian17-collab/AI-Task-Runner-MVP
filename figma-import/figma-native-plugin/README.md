# Figma Native Import Plugin

This local Figma plugin recreates the AI Task Runner UI as native editable Figma layers.

## How To Use

1. Open the Figma desktop app.
2. Open a design file.
3. Go to `Plugins -> Development -> Import plugin from manifest...`.
4. Select:
   `E:\Codex-workespace\Project1\figma-import\figma-native-plugin\manifest.json`
5. Run `Plugins -> Development -> AI Task Runner UI Importer`.

The generated result is made from native Frames, Rectangles, Ellipses, and Text layers, so it is much easier to edit than the SVG import.
