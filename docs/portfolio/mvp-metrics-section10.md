**MVP 小样本观察（n=3，待真实试用数据替换）**

| 指标 | 当前观察 | 设计含义 |
|---|---:|---|
| 首步启动率 | 3/3 | “当前一步”能帮助用户从模糊任务进入行动入口 |
| 当前小步完成 / 产出率 | 3/3 | 降低质量门槛后，用户更容易形成第一个可继续修改的产出 |
| 卡点触发次数 | 3 次 | 卡点不是异常，而是任务推进体验必须处理的核心场景 |
| fallback 后继续率 | 2/3 | 降阻力步骤对继续推进有帮助，但恢复策略仍需更多真实任务验证 |
| 从输入任务到进入第一步 | 平均 1.3 次交互 | 大多数任务不需要复杂表单，少量信息不足任务需要一次澄清 |
