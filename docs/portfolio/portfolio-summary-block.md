**Summary**

AI Task Runner 解决的是个人任务推进中的“知道要做，但不知道现在怎么开始”的问题，尤其适用于求职、学习和创作这类模糊、高压力任务。

我的核心设计决策是让 AI 不先生成完整计划，而是始终围绕一个低阻力、可执行的“当前一步”推进；当信息不足或用户卡住时，再通过澄清和降阻力恢复把用户带回行动。

当前 MVP 已验证从任务输入、StepCard 执行、下一步生成、信息澄清到卡点恢复的核心闭环；访谈样本也支持“用户缺的不是更多计划，而是当前动作”的判断。

角色：产品定义、AI 产品体验设计、交互设计、前端原型与后端 AI 调用。技术栈：React + Vite / Express / DeepSeek API / localStorage。
