# AI Task Runner Logo Exploration

This folder contains the first-round logo exploration for AI Task Runner.

## Files

- `01-current-step-mark.svg` - recommended app icon direction
- `02-low-friction-path.svg` - motion and low-friction task start
- `03-resistance-recovery.svg` - blocked-state recovery
- `04-step-card-grid.svg` - UI-native StepCard grid
- `05-context-loop.svg` - short-term context and confirmation loop
- `06-guided-node.svg` - guided decision/node system
- `ai-task-runner-logo-showcase.html` - comparison page

## Brand Notes

AI Task Runner should be framed as an AI product experience MVP that helps users move from "stuck" to a current executable step. The logo should avoid overclaiming full autonomous-agent behavior.

Recommended first direction: `01-current-step-mark.svg`.
